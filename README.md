# vuepress-plugin-plausible-analytics

VuePress plugin to add analytics from plausible.io

## Install

Run command:

```bash
yarn add -D vuepress-plugin-plausible-analytics
```

## Usage

Add in file `.vuepress/config.js`:

```json
module.exports = {
  plugins: [
    "vuepress-plugin-plausible-analytics",
  ]
}
```
